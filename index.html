Datenschutzerklärung für TrackyIO

Diese Datenschutzerklärung informiert Sie über die Verarbeitung personenbezogener Daten in der App TrackyIO.

1. Verantwortlicher

Danyal Yayla
E-Mail: deine_email@example.com

2. Erhobene Daten

  • Gewichtsdaten
  • Trainingseinträge
  • Lebensmittel- und Barcode-Scans
  • Schrittdaten (über Apple Health)
  • Optional: iCloud-Synchronisation

3. Zweck der Verarbeitung

Die Daten werden ausschließlich zur Nutzung der App gespeichert, zur Berechnung von Fortschritt, Training, Ernährungszielen und Empfehlungen.

4. Speicherung der Daten

Alle Daten werden lokal auf dem Gerät gespeichert. Optional können Sie die Daten über iCloud synchronisieren lassen.

5. Weitergabe an Dritte

TrackyIO gibt keine personenbezogenen Daten an Dritte weiter.

6. HealthKit

TrackyIO verwendet HealthKit ausschließlich zum Lesen von Schritten und Aktivitätsdaten. Diese Daten werden nur innerhalb der App verwendet und nicht an Server oder Dritte gesendet.

7. Rechte der Nutzer

Sie können Ihre Daten jederzeit löschen, indem Sie die App entfernen oder in den App-Einstellungen „Daten zurücksetzen“ wählen.
